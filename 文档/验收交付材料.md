## 三、演示流程脚本（讲解用）

### 3.1 整体流程（20分钟）

```
【开场 - 2分钟】
各位老师好，我今天演示的项目是"校园万事达互助众包任务平台"。

【功能介绍 - 3分钟】
这是一个面向高校学生的互助平台，主要解决校园生活中"找人帮忙"的需求。

核心功能包括：
1. 任务发布与承接
2. 资金担保机制
3. 信用评价体系
4. 实时通知推送
5. 管理员后台

【演示 - 15分钟】

1. 用户注册登录（3分钟）
   - 演示需求方注册
   - 演示接单方注册
   - 演示登录

2. 发布任务（3分钟）
   - 登录需求方账号
   - 发布任务（奖励10元）
   - 展示资金冻结（10+0.5服务费）

3. 接单与完成（4分钟）
   - 切换接单方账号
   - 接单操作
   - 提交凭证
   - 切换回需求方
   - 验收确认
   - 展示资金结算

4. 评价功能（2分钟）
   - 发布5星评价
   - 展示信用分变化

5. 管理后台（3分钟）
   - 登录管理员
   - 查看统计数据
   - 争议处理演示

【总结 - 2分钟】
以上就是校园万事达平台的核心功能演示，谢谢大家！
```
