# 用户操作手册

## 一、系统简介

校园"万事达"互助众包任务平台是一个为高校学生提供互助任务发布与接单的在线平台。用户可以通过平台发布互助需求（如取快递、代买东西等），其他用户可以承接任务并获得报酬。

### 1.1 主要功能

- 用户注册与登录
- 发布互助任务
- 承接并完成任务
- 账户充值与资金管理
- 双向评价与信用评分
- 实时通知推送
- 争议处理（管理员）

---

## 二、账号管理

### 2.1 注册账号

**操作步骤**：

1. 打开平台首页，点击"注册"按钮
2. 填写注册信息：
   - 用户名：2-20个字符
   - 手机号：11位数字
   - 密码：6位以上
   - 角色选择：
     - **需求方**：需要发布任务的用户
     - **接单方**：想要接任务赚取报酬的用户
3. 点击"注册"按钮完成注册

**注意事项**：
- 手机号将作为登录账号，请确保填写正确
- 角色选择后可以在个人信息中修改
- 密码请妥善保管，不要泄露给他人

### 2.2 登录账号

**操作步骤**：

1. 打开平台首页，点击"登录"按钮
2. 输入手机号和密码
3. 点击"登录"按钮

**注意事项**：
- 连续5次密码错误将被锁定30分钟
- 忘记密码请联系管理员

### 2.3 修改密码

**操作步骤**：

1. 登录后点击右上角头像进入"个人中心"
2. 点击"修改密码"
3. 输入原密码和新密码
4. 点击"确认修改"

## 三、任务大厅

### 3.1 浏览任务

**操作步骤**：

1. 首页默认为"任务大厅"
2. 可以通过以下方式筛选任务：
   - 状态筛选：待接单/进行中/已完成
   - 分类筛选：校园服务/生活互助/学习帮助/其他
3. 点击任务卡片可以查看任务详情

### 3.2 发布任务（需求方）

**操作步骤**：

1. 登录并确认账号角色为"需求方"
2. 点击"发布任务"按钮
3. 填写任务信息：
   - 任务标题：简洁明了，不超过50字
   - 任务描述：详细说明任务要求和注意事项
   - 任务分类：选择合适的分类
   - 奖励金额：接单方完成任务后可获得的报酬
   - 截止时间：任务最晚完成时间
   - 任务地点：详细地址
   - 联系方式：手机号或其他联系方式
4. 点击"确认发布"

**注意事项**：
- 发布任务需要冻结奖励金额+5%服务费，请确保账户余额充足
- 奖励金额最低1元
- 任务发布后可以在"我的任务"中查看和管理

### 3.3 接单任务（接单方）

**操作步骤**：

1. 登录并确认账号角色为"接单方"
2. 在任务大厅选择想要承接的任务
3. 点击"接单"按钮
4. 确认接单

**注意事项**：
- 每人每天最多接3个任务
- 接单前请确保能在截止时间前完成任务
- 信用分低于60分无法接单
- 无法接自己发布的任务

### 3.4 提交完成凭证（接单方）

**操作步骤**：

1. 登录后进入"我的任务"
2. 找到"进行中"状态的任务
3. 点击"提交凭证"按钮
4. 上传任务完成的证明材料（图片或文字说明）
5. 点击"确认提交"

**注意事项**：
- 凭证将发送给需求方进行验收
- 提交后任务将进入"待验收"状态

### 3.5 验收任务（需求方）

**操作步骤**：

1. 登录后进入"我的任务"
2. 找到"待验收"状态的任务
3. 点击"查看详情"确认任务完成情况
4. 如确认完成，点击"确认完成"按钮
5. 如有问题，点击"发起争议"按钮

**注意事项**：
- 确认完成后，冻结资金将转入接单方账户
- 如有问题请先与接单方沟通

## 四、账户管理

### 4.1 账户充值

**操作步骤**：

1. 登录后点击右上角头像进入"个人中心"
2. 点击"账户充值"
3. 输入充值金额
4. 点击"确认充值"

**注意事项**：
- 充值金额将直接添加到可用余额
- 充值用于发布任务或提现

### 4.2 查看账户明细

**操作步骤**：

1. 登录后进入"个人中心"
2. 点击"账户明细"
3. 可以查看：
   - 可用余额：可以使用的金额
   - 冻结金额：任务担保中被冻结的金额
   - 累计收入：所有任务获得的报酬
   - 累计支出：发布任务支出的金额
4. 可以按类型筛选交易记录

---

## 五、评价管理

### 5.1 发布评价

**操作步骤**：

1. 任务完成后自动进入评价页面
2. 选择评分（1-5星）
3. 填写评价内容（可选）
4. 点击"提交评价"

**评分规则**：
- 5星：非常好，服务到位
- 4星：比较好，满意
- 3星：一般般
- 2星：较差，不满意
- 1星：非常差，投诉

**信用分更新规则**：
- 5星好评：+2分
- 4星好评：+1分
- 3星中评：+0分
- 1-2星差评：-5分
- 信用分范围：0-100分

### 5.2 查看评价

**操作步骤**：

1. 登录后进入"个人中心"
2. 点击"我的评价"
3. 可以查看收到的所有评价
4. 也可以查看发出的评价

---

## 六、通知中心

### 6.1 查看通知

**操作步骤**：

1. 登录后点击右上角铃铛图标
2. 进入通知列表
3. 点击通知可以查看详情

**通知类型**：
- 任务被接单
- 任务已提交
- 任务已完成
- 收到新评价
- 争议处理结果
- 系统通知

### 6.2 标记已读

**操作步骤**：

1. 在通知列表中点击单条通知
2. 该通知自动标记为已读
3. 如需全部标记，点击"全部已读"按钮

---

## 七、争议处理

### 7.1 发起争议

**操作步骤**：

1. 任务状态为"待验收"时
2. 点击"发起争议"按钮
3. 填写争议原因（必填）
4. 点击"提交争议"

**注意事项**：
- 争议提交后任务将进入"争议中"状态
- 管理员将在1-3个工作日内处理

### 7.2 争议处理结果

争议处理完成后，系统将通知双方结果：

- **支持需求方**：冻结资金全额退还需求方
- **支持接单方**：冻结资金转入接单方账户
- **部分支持**：按比例分配资金

### 8.1 个人信息

可以查看和修改以下信息：
- 用户名
- 手机号（不可修改）
- 角色
- 信用分

### 8.2 我的任务

- 全部任务
- 我发布的
- 我承接的
- 待验收
- 已完成
- 争议中

## 九、常见问题

### 9.1 忘记密码怎么办？

请联系管理员，提供手机号和身份证明，管理员可以协助重置密码。

### 9.2 账户余额不足怎么办？

可以通过"账户充值"功能充值，充值后即可发布任务。

### 9.3 任务过期了怎么办？

如果任务超过截止时间且接单方未提交凭证，任务将自动取消，资金解冻。

### 9.4 被恶意差评怎么办？

可以在"意见反馈"中向管理员投诉，管理员有权删除恶意评价。

### 9.5 如何提高信用分？

通过获得好评可以提高信用分。保持良好的服务态度，及时完成任务是获得好评的关键。
