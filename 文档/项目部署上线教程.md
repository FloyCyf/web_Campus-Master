## 一、环境要求

| 软件 | 版本 | 说明 |

| MySQL | 8.0+ | 数据库 |

## 二、后端启动

### 2.1 导入数据库

```bash
# 登录MySQL
mysql -u root -p

# 创建数据库
CREATE DATABASE campus_master DEFAULT CHARACTER SET utf8mb4;

# 使用数据库
USE campus_master;

# 导入表结构
SOURCE e:/web/backend/src/main/resources/schema.sql;

# 导入初始数据
SOURCE e:/web/backend/src/main/resources/data.sql;
```

### 2.2 修改数据库配置

编辑 `e:\web\backend\src\main\resources\application.yml`：

```yaml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/campus_master?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: Zdx962436510!
    driver-class-name: com.mysql.cj.jdbc.Driver
```

### 2.3 启动后端

```bash
cd e:\web\backend
mvn spring-boot:run
```

或先打包再运行：
```bash
cd e:\web\backend
mvn clean package -DskipTests
java -jar target/campus-master-1.0.0.jar
```

**后端启动地址**：http://localhost:8080

**验证后端是否启动成功**：
```bash
# 测试登录接口
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800000000","password":"admin123"}'
```

## 三、前端启动

### 3.1 安装依赖

```bash
cd e:\web
npm install
```

### 3.2 启动前端开发服务器

```bash
npm run dev
```

**前端启动地址**：http://localhost:3000

### 3.3 API代理配置

`vite.config.js` 已配置API代理：

```javascript
proxy: {
  '/api': {
    target: 'http://localhost:8080',
    changeOrigin: true
  }
}
```

前端请求 `/api/*` 会自动代理到后端 `http://localhost:8080/api/*`。

## 六、常见问题处理

### 6.1 端口被占用

**问题**：`Web server failed to start. Port 8080 was already in use.`

**解决**：
```bash
# 查找占用8080端口的进程
netstat -ano | findstr 8080

# 终止进程（将 <PID> 替换为实际进程ID）
taskkill /F /PID <PID>
```

### 6.2 数据库连接失败

**问题**：`Cannot get connection from pool`

**检查项**：
1. MySQL服务是否启动
2. 数据库配置中的用户名和密码是否正确
3. 数据库 `campus_master` 是否已创建
4. 防火墙是否允许3306端口

### 6.3 登录失败

**问题**：`手机号或密码错误`

**检查项**：
1. 确保使用手机号登录（非用户名）
2. 密码是否正确（区分大小写）
3. 后端服务是否正常运行
4. 数据库中是否存在该用户

### 6.4 Token失效

**问题**：`登录已失效，请重新登录`

**原因**：前端请求未携带或携带了无效的JWT token

**解决**：确保登录成功后保存token，并在请求时携带：
```javascript
// 登录成功后保存token
localStorage.setItem('token', response.data.token);

// 请求时携带token
axios.get('/api/notification/list', {
  headers: {
    'Authorization': 'Bearer ' + localStorage.getItem('token')
  }
});
```
