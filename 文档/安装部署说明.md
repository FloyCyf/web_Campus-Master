# 安装部署说明

## 一、环境准备

### 1.1 软件要求

| 软件 | 版本 | 说明 |
|------|------|------|
| JDK | 17+ | 后端运行环境 |
| Node.js | 18+ | 前端构建需要 |
| npm | 9+ | 前端包管理 |
| MySQL | 8.0+ | 数据库 |

---

## 二、后端部署

### 2.1 安装JDK 17

**Windows**：

1. 下载JDK 17：https://adoptium.net/
2. 安装并配置环境变量：
   ```
   JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-17.0.9.9-hotspot
   PATH = %JAVA_HOME%\bin
   ```

2. 验证安装：
   ```cmd
   java -version
   ```

**Linux**：

```bash
# Ubuntu
sudo apt update
sudo apt install openjdk-17-jdk

# CentOS
sudo yum install java-17-openjdk java-17-openjdk-devel
```

### 2.2 安装MySQL 8

**Windows**：

下载MySQL Installer：https://dev.mysql.com/downloads/installer/

**Linux (Ubuntu)**：

```bash
sudo apt update
sudo apt install mysql-server
sudo systemctl start mysql
sudo systemctl enable mysql
```

**Linux (CentOS)**：

```bash
sudo yum install mysql-server
sudo systemctl start mysqld
sudo systemctl enable mysqld
```

### 2.3 创建数据库

```sql
-- 登录MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE campus_master DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2.4 导入数据库表结构

```bash
# 进入后端目录
cd e:\web\backend

# 命令行导入
mysql -u root -p campus_master < src/main/resources/schema.sql
mysql -u root -p campus_master < src/main/resources/data.sql
```

或使用MySQL客户端工具（如Navicat、DBeaver）导入 `schema.sql` 和 `data.sql`。

### 2.5 配置数据库连接

编辑 `src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/campus_master
    username: root
    password: Zdx962436510!
```

### 2.6 启动后端

```bash
cd e:\web\backend

# 使用Maven启动
mvn spring-boot:run
```

后端启动成功后会显示：
```
Started CampusMasterApplication in x seconds
```

访问地址：http://localhost:8080

## 三、前端部署

### 3.1 安装Node.js

**Windows**：

1. 下载LTS版本：https://nodejs.org/
2. 安装，勾选"Add to PATH"
3. 验证：`node -v`

**Linux**：

```bash
# 使用nvm安装
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
nvm use 18
```

### 3.2 安装依赖

```bash
cd e:\web
npm install
```

### 3.3 启动开发服务器

```bash
npm run dev
```

前端启动成功后会显示：
```
VITE v5.x.x  ready in xxx ms

➜  Local: http://localhost:3000/
```

访问地址：http://localhost:3000

## 五、常见问题

### 5.1 数据库连接失败

```
症状：java.sql.SQLException: Access denied
```

**解决**：
1. 检查用户名密码是否正确
2. 检查MySQL服务是否启动
3. 验证命令：`mysql -u root -p`

### 5.2 端口被占用

```
症状：Address already in use
```

**解决**：

Windows：
```cmd
netstat -ano | findstr 8080
taskkill /F /PID <PID号>
```

Linux：
```bash
lsof -i:8080
kill -9 <PID号>
```

### 5.3 前端无法访问API

**解决**：
1. 检查后端服务是否启动在8080端口
2. 检查防火墙是否阻止了端口
3. 检查vite.config.js代理配置是否正确
